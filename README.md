# Modakmay

Authentic ukhdiche modak, kothimbir vadi, and traditional Maharashtrian snacks.

## Quick start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Before going live

**Read [SETUP.md](./SETUP.md)** — step-by-step guide for non-technical setup, deployment, and Google Business Profile linking.

## Edit business details

All contact info lives in one file:

```
src/config/site.ts
```

## Tech stack

- Next.js 15
- Tailwind CSS 4
- Framer Motion
- Lucide Icons
- Deployed on Vercel

## Project structure

```
src/
├── app/           → Pages and layout
├── components/    → Website sections
└── config/
    └── site.ts    → ⭐ Edit your business details here
public/            → Your photos go here
```
